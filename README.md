# Consark.ai QA Automation Assignment - File.io Workflow
Task: To automate file multi file upload taks for file.io using playwright with typescript strictly following POM iplementation and helper fucntions.

This repository contains an enterprise-grade end-to-end automation testing suite targeting the `File.io` file-sharing workflow. Built with **Playwright** and **TypeScript**, the framework follows the **Page Object Model (POM)** pattern to ensure high scalability, clean isolation of locators, and resilient handling of asynchronous UI states.

## 📂 Project Structure
As per the technical mandate, the project structure is explicitly organized as follows:
- `tests/` — End-to-end test execution spec suites.
- `pages/` — Page Object classes containing locators and element interaction strategies.
- `utils/` — Custom helper classes for processing dynamic file system stream validations.
- `test-data/` — Isolated staging assets (.csv, .png, .pdf) utilized for upload criteria.
